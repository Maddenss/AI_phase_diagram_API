# API для модели машинного обучения

Это API для модели машинного обучения, созданное с использованием библиотеки FastAPI. Оно позволяет загружать данные, обучать модель XGBoost, строить графики (ROC-AUC, важность признаков и SHAP) и многое другое.

## Установка

Для установки зависимостей создайте виртуальное окружение и активируйте его. Затем установите все необходимые библиотеки из файла `requirements.txt`.
